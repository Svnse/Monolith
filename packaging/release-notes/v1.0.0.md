First public v1 source release of the Monolith desktop workstation: chat and
model runtime, tool/skill execution, persistence, optional media, CONNECT/Matrix,
Workshop/Monoline integration, and experimental cognition substrates.

**Easiest install:** download `MonolithInstaller_1.0.0.exe` below and run it — it
sets up Monolith with its own bundled Python and a Start Menu/desktop shortcut.
No Python required on your machine. (Windows 10/11. The installer downloads
dependencies, so it needs internet.) Prefer source? `git clone` + `install.bat`
as described in the README.

## Added
- Windows installer (`MonolithInstaller_*.exe`): bundled CPython 3.11 runtime,
  isolated install under `%LOCALAPPDATA%\Programs\Monolith`, Start Menu and
  desktop shortcuts, self-healing first-launch dependency setup. User data
  stays in `%APPDATA%\Monolith` and is never touched by uninstall.
- Public release manifest and verification workflow.
- Structured GitHub issue forms, pull-request template, and Windows CI.
- Public installation, configuration, architecture, security, support,
  provenance, third-party, and known-issues documentation.

## Changed
- Root `README.md` now describes the current capability/maturity boundary,
  tested platform, data/network behavior, and exact test status.
- `pyproject.toml` is the dependency source of truth, with separate base,
  files, local-LLM, vision, audio, Matrix, and developer groups.
- `install.bat` validates Python 3.11, uses the environment interpreter,
  consumes package metadata, stops on failure, and runs a smoke check.
- `start.bat` anchors to its own directory and uses a conservative public
  profile without autonomous, identity-mutation, or network flags.

## Security
- CONNECT all-interface startup is blocked unless `MONOLITH_AGENT_TOKEN` is non-empty.
- CONNECT `/events` applies the same authentication boundary as other protected routes.
- CONNECT has no TLS/RBAC and remains unsupported for remote exposure.

## Verification
- Default test profile (`[dev,files,matrix]`, no torch): 3102 passed, 60 skipped,
  1 xfailed, 0 failed. Full-extras baseline: 3,173 collected / 3,113 passed.
  See `KNOWN_ISSUES.md`.
- Windows 10/11 + Python 3.11 is the tested platform; Linux/macOS unverified.

`v1.0.0` establishes the stable public core release line. It does not imply
production multi-user hosting, stable experimental schemas, or complete
optional workspaces. The pre-v1 history remains available on the
`legacy-alpha` branch.
